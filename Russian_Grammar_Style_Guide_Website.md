# Russian Grammar & Style Guide for Website Translation

## Purpose

This guide defines the grammatical and stylistic rules that AI should
follow when generating or translating Russian website content. The goal
is to produce text that sounds like it was written by a professional
native Russian copywriter---not as a literal translation.

------------------------------------------------------------------------

# 1. General Writing Style

-   Write in modern standard Russian.
-   Use a calm, professional and trustworthy tone.
-   Write naturally and fluently.
-   Avoid marketing hype.
-   Avoid literal translations from German or English.

------------------------------------------------------------------------

# 2. Sentence Structure

-   One idea per sentence.
-   Prefer short to medium-length sentences.
-   Avoid unnecessarily complex subordinate clauses.
-   Structure:
    -   Statement
    -   Explanation
    -   Benefit

Example:

> Английский --- это инструмент для работы, общения и самовыражения.

------------------------------------------------------------------------

# 3. Verb Tense

Use the **present tense** in almost all website texts.

Examples:

-   Я помогаю...
-   Студенты учатся...
-   Мы работаем...

------------------------------------------------------------------------

# 4. Active Voice

Prefer active voice whenever possible.

Preferred:

> Я помогаю взрослым специалистам...

Instead of:

> Обучение осуществляется...

------------------------------------------------------------------------

# 5. Verbs

Prefer concrete, meaningful verbs.

Recommended:

-   помогать
-   работать
-   говорить
-   использовать
-   применять
-   понимать
-   развивать
-   решать
-   преодолевать
-   учиться

Avoid bureaucratic wording such as:

-   осуществлять
-   производить
-   реализовывать (unless necessary)

------------------------------------------------------------------------

# 6. Nouns

Prefer concrete nouns.

Examples:

-   работа
-   практика
-   общение
-   опыт
-   язык
-   речь
-   результат

------------------------------------------------------------------------

# 7. Adjectives

Use adjectives only when they add information.

Preferred:

-   международный
-   профессиональный
-   практический
-   уверенный
-   реальный

Avoid exaggerated adjectives:

-   лучший
-   идеальный
-   невероятный
-   революционный
-   уникальный

------------------------------------------------------------------------

# 8. Dash

Always use the Russian em dash:

---

Example:

> Английский --- это инструмент.

Never replace it with a hyphen (-).

------------------------------------------------------------------------

# 9. Lists

All bullet points must have the same grammatical structure.

Correct:

-   прохождение собеседований
-   взаимодействие с коллегами
-   релокация
-   решение рабочих задач

Do not mix infinitives, nouns and complete sentences.

------------------------------------------------------------------------

# 10. Cases

Always use correct Russian cases.

Examples:

-   для работы
-   для карьеры
-   с клиентами
-   с коллегами
-   в международной среде
-   за границей
-   без языкового барьера

------------------------------------------------------------------------

# 11. Pronouns

Avoid repeating "Я" too often.

Alternate with:

-   Мой подход...
-   В основе моей работы...
-   Особое внимание...
-   На занятиях...

------------------------------------------------------------------------

# 12. Participles

Avoid excessive participle constructions.

Preferred:

> Люди, которые работают...

Instead of:

> Работающие специалисты...

------------------------------------------------------------------------

# 13. Vocabulary

Use modern Russian.

Allowed international terms:

-   релокация
-   адаптация
-   кейс
-   Cambridge CELTA

Avoid unnecessary English words.

------------------------------------------------------------------------

# 14. Tone

The text should sound:

-   professional
-   competent
-   calm
-   approachable
-   trustworthy

Never:

-   aggressive
-   overly emotional
-   salesy
-   exaggerated

------------------------------------------------------------------------

# 15. Headings

Use short headings (2--5 words).

Examples:

-   Обо мне
-   Мой подход
-   Для кого курс
-   Как проходят занятия
-   Что вы получите

------------------------------------------------------------------------

# 16. Paragraphs

Maximum 3--4 sentences per paragraph.

Leave whitespace between paragraphs.

------------------------------------------------------------------------

# 17. Translation Rules

When translating:

-   Never translate word for word.
-   Translate meaning, not grammar.
-   Rewrite sentences if necessary to sound natural in Russian.
-   Preserve the original intent and tone.
-   Remove unnatural German or English sentence structures.

------------------------------------------------------------------------

# 18. AI Quality Checklist

Before returning any Russian text, verify:

-   Correct grammar
-   Correct cases
-   Natural word order
-   Active voice where possible
-   Consistent bullet lists
-   Natural vocabulary
-   No literal translation
-   Professional native style
-   Clear and readable formatting
-   Suitable for an educational website

------------------------------------------------------------------------

# Goal

Every Russian text should read as if it were written by an experienced
native Russian copywriter for a premium educational website.
